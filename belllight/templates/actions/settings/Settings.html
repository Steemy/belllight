Для вывода css и js на сайте, вызовите в вашем шаблоне index.html перед закрывающим тегом </head> {if $wa->shop}{shopBelllightPlugin::head()}{/if}.
Для вывода самой формы, вызовите в вашем шаблоне index.html перед закрывающим тегом </body> {if $wa->shop}{shopBelllightPlugin::display()}{/if}.

Для вызова формы надо прописать тег который будет вызывать форму, к примеру пропишите в шаблоне <a href="#" class="reqCall">Заказать звонок</a>
reqCall - это класс который будет вызывать форму надо прописать в настройках плагина «Селектор кнопки вызова формы»: - .reqCall
